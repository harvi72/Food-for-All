# 🍲 Food For All

Food For All is a web platform that helps reduce food waste by connecting food donors with people in need.

## 🚀 Features

- Donate surplus food
- View available donations
- Live location map
- Detect donor location
- Filter food or money donations
- Animated UI with modern design
- Donation cards with details

## 🛠 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Libraries:
- Leaflet.js (Map)
- OpenStreetMap API (Location)
